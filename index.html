FAQs

What is Frontend Mentor, and how will it help me? +

Frontend Mentor offers realistic coding challenges to help developers improve their frontend coding skills with projects in HTML, CSS, and JavaScript. It's suitable for all levels and ideal for portfolio building.

How do I get started? +

Choose a challenge, read the instructions carefully, and download the starter code. Next, start coding! Once you're done, submit your solution on the platform for feedback from the Frontend Mentor community.

What should I do if I need help? +

If you need help, ask the community on the platform's Discord channel. You can also browse the comments section on each challenge to see solutions and advice from other developers.

How can I give feedback or suggest a new feature? +

You can leave feedback on the platform's Discord channel or contact the team directly via email. If you have a feature request, you can also submit it on the platform.

Can I use Frontend Mentor for free? +

Yes, Frontend Mentor offers both free and premium coding challenges. The free option provides access to a range of projects suitable for all skill levels.

Can I use Frontend Mentor projects in my portfolio? +

Yes, you can use projects completed on Frontend Mentor in your portfolio. It's an excellent way to showcase your skills to potential employers!

How can I get help if I'm stuck on a Frontend Mentor challenge?

+

The best place to get help is inside Frontend Mentor's Discord community. There's a help channel where you can ask questions and seek support from other community members.
